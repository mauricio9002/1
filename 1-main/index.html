lei geral de proteção de dados (lgpd)

a lei geral de proteção de dados (lgpd) é a legislação brasireila que regula as atividades de tratamento de dados pessoais. Ela tem como objetivo protejer os direitos fundamentais de liberdade e de privacidade, bem como bem como livre dsemvolvimento da personalidade da pessoa natural

principais pontos da lgpd:

  • transparencia: As organizaçoes devem fornecer informação claras sobre a coleta, uso e armazenamennto dos dados pesoais.
  • conentimento: O conentimento do titular deve ser obtido para o tratamento dados, salvo exceções previstas na lei.
  • finalidade: Os dados devem ser coletados para propósitos específicos, explícitos e legitimos.
  • segurança: As organizaçoes devem garantir a proteção dos dados pessoais contra aesos não autorizados e situações acidentais ou ilicitas
  • direitos dos titulares:Os titulares dos dados têm o direito de acesar, corrigir, eliminar e porbilizarseus dados peoais.
